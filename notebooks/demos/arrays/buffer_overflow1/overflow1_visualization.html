Buffer Overflow Visualizer: buffer_overflow1.cpp

This models a common classroom stack layout where password[8] and social_security[12] sit next to each other in memory. Typing more than 7 characters causes cin >> password to keep writing bytes past the end of password, corrupting social_security.

Program would print:
your SSN is 
char social_security[] = "123-45-6789";
char password[8];

cout << "Enter your password: ";
cin >> password;  // no bounds check
cout << "your SSN is " << social_security << endl;

Note: real stack layout can vary by compiler, optimization, and platform. This is a teaching visualization aligned with the demo's intended behavior.
